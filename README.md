# decimal62_converters
Decode decimal62 to decimal10

## Usage

```
decimal10 = 12345
decimal62 = 3d7
```

### Rscript

```
$ Rscript decode_decimal62.r
[1] "0"
[1] "3d7"
[1] 0
[1] 12345
```

### Python

```
$ python convert_decimal62.py
0
3d7
0
12345
```

### Java

Compile

```
$ javac Decimal62Converter.java
```

Execute

```
$ java Decimal62Converter
0
3d7
0
12345
```
